# AI-Powered UGC Ad Content Generator

## Project Overview

This project demonstrates a reusable AI prompt framework designed to generate authentic User-Generated Content (UGC) style advertising content for businesses and brands.

The system generates multiple hooks, ad scripts, call-to-actions, and social media captions that are ready to use for short-form content platforms such as Instagram Reels, YouTube Shorts, and Facebook Ads.

For this project, the prompt was tested using a D2C skincare business selling an acne control serum.

---

## Selected Business

**Business Type:** D2C Skincare Business

**Product:** Acne Control Serum

**Target Audience:** Men and women aged 18–35 with acne-prone skin

**Customer Problem:** Frequent breakouts, acne marks, and uneven skin texture

**Key Benefit:** Supports clearer-looking skin and helps maintain a simple skincare routine

---

## Prompt Design

The prompt was designed to generate a complete UGC Ad Content Pack while remaining reusable for different businesses and industries.

The framework accepts business-specific inputs such as:

* Business Type
* Product or Service
* Target Audience
* Customer Pain Point
* Key Benefit
* Unique Selling Point (USP)
* Platform
* Brand Tone

These variables can be replaced to generate content for different brands and industries.

---

## Prompt Logic

The prompt follows a structured content generation workflow:

### 1. Hook Generation

Creates multiple hook variations for the same product using different psychological angles:

* Problem-Based Hooks
* Curiosity Hooks
* Personal Experience Hooks
* Transformation Hooks
* Contrarian Hooks

This ensures multiple creative directions for testing ad performance.

---

### 2. UGC Ad Script Generation

Each ad script follows a proven marketing framework:

**Hook → Problem → Discovery → Solution → Benefit → CTA**

The scripts are written from different customer perspectives:

* First-Time User
* Repeat Customer
* Friend Recommendation
* Before-and-After Experience
* Problem-Solver Testimonial

This creates realistic user-generated content that feels authentic and relatable.

---

### 3. Platform Adaptation

The prompt generates content optimized for:

* Instagram Reels
* YouTube Shorts
* Facebook Ads

Each platform requires different pacing, engagement style, and audience expectations.

---

### 4. Conversion Optimization

The system generates multiple CTA styles:

* Direct CTAs
* Soft CTAs
* Urgency-Based CTAs
* Curiosity-Based CTAs
* Trust-Based CTAs

This improves flexibility for different marketing objectives.
